I got sick of the default Windows slideshow screensaver being crap so I made my own one.

Supports multiple monitors, doesnt shuffle through randomly but iterates from a random position.

Also remembers what image you were looking at if you hit Ctrl will show you the image so you can share it or stare at it if you want.

Release builds coming soon (tm) rn just build it yourself with dotnet build and put the .scr in System32, if you nag me I'll make a github actions workflow for it.
